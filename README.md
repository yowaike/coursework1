# Электронный дневничок

Курсовая работа - система учёта успеваемости в школе.

## Стек технологий
- Бэкенд: Python 3.14 + FastAPI
- Фронтенд: React + Jinja2 (многостраничное приложение)
- База данных: PostgreSQL 18
- Авторизация: JWT в cookies + bcrypt

## Кто может пользоваться
- **Завуч** - управляет учителями, учениками, классами, смотрит аналитику
- **Учитель** - выставляет оценки по своему предмету, видит расписание
- **Ученик** - видит свой дневник, табель с оценками и заметки

## Как запустить локально
1. Установите зависимости:
pip install -r backend/requirements.txt

2. Создайте файл `.env` с настройками PostgreSQL (можно скопировать из `.env.example`)

3. Инициализируйте базу данных:
python backend/init_db.py

4. Запустите сервер разработки:
uvicorn backend.app.main:app --reload

5. Откройте в браузере: `http://127.0.0.1:8000`

**Тестовые аккаунты** (создаются при инициализации):
- `admin@school.ru` / `admin123` - завуч
- `teacher@school.ru` / `teacher123` - учитель
- `student@school.ru` / `student123` - ученик
