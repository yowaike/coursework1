# Электронный дневничок

Курсовая работа – система учёта успеваемости в школе.

## Стек технологий
- **Бэкенд:** Python 3.14 + FastAPI  
- **Фронтенд:** Jinja2 + React 18 (гибридный подход, React‑компоненты через CDN)  
- **База данных:** PostgreSQL 18  
- **Аутентификация:** JWT в HttpOnly cookies, хеширование паролей bcrypt

## Роли в системе
- `admin` – завуч  
- `teacher` – учитель  
- `student` – ученик

## Возможности по ролям

### Завуч
- Управление учителями, учениками, классами, предметами  
- Управление расписанием  
- Выставление четвертных оценок, переопределение годовых  
- Просмотр аналитики (5 отчётов)  
- Просмотр аудит‑логов  
- Изменение глобальных полей (школа, город, учебный год)
- Заметки (любым ученикам)

### Учитель
- Просмотр своего расписания  
- Список учеников по классам  
- Выставление текущих оценок (ДЗ, КР, ОТВ, СР, ТЕСТ)  
- Просмотр текущих оценок своих учеников  
- Заметки своим ученикам

### Ученик
- Дневник с карточками предметов  
- Детализация всех оценок по предмету 
- Табель успеваемости  
- Расписание класса  
- Личные заметки + заметки от учителя/завуча

##  Локальный запуск

1. **Установите зависимости**  
   ```bash
   pip install -r backend/requirements.txt
   ```

2. **Настройте базу данных**  
   Создайте файл `.env` в папке `backend` (пример в `.env.example`):  
   ```env
   DATABASE_URL=postgresql://postgres:1234@127.0.0.1:5432/school_diary
   ```

3. **Инициализируйте базу данных**  
   ```bash
   cd backend
   python init_db.py
   ```

4. **Заполните демо‑данными (оценки, расписание)**  
   ```bash
   python populate_test_data.py
   ```

5. **Запустите сервер**  
   ```bash
   python -m uvicorn app.main:app --reload
   ```

6. **Откройте в браузере**  
   ```
   http://127.0.0.1:8000
   ```
```

**Тестовые аккаунты**:
- `admin@school.ru` / `admin123` - завуч
- `sidorov@school.ru` / `teacher123` - учитель по русскому языку
