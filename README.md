# Электронный дневничок

Курсовая работа - система учёта успеваемости в школе.

## Стек технологий
- Бэкенд: Python 3.14 + FastAPI
- Фронтенд: React 18 через CDN, React SPA с серверным API
- База данных: PostgreSQL 18
- Авторизация: JWT в HttpOnly cookies + bcrypt

## Роли в коде
- **admin** - роль завуча в коде
- **teacher** - учитель
- **student** - ученик

## Кто может пользоваться
- **Завуч** - управляет учителями, учениками, классами, предметами, расписанием и смотрит аналитику
- **Учитель** - выставляет оценки по своему предмету, видит свое расписание и список своих учеников
- **Ученик** - видит свой дневник, табель и заметки

## Как запустить локально
1. Установите зависимости:
pip install -r backend/requirements.txt

2. Создайте файл `.env` с настройками PostgreSQL (можно скопировать из `.env.example`)

3. Инициализируйте базу данных:
python backend/init_db.py

4. Запустите сервер разработки:
uvicorn backend.app.main:app --reload

5. Откройте в браузере: `http://127.0.0.1:8000`
