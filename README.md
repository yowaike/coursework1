# Электронный дневничок

Курсовая работа - система учёта успеваемости в школе.

## Стек технологий
- Бэкенд: Python 3.14 + FastAPI
- Фронтенд: React + Jinja2 (многостраничное приложение)
- База данных: PostgreSQL 18
- Авторизация: JWT в cookies + bcrypt

## Кто может пользоваться
- Завуч - управляет учителями, учениками, классами, смотрит аналитику
- Учитель - выставляет оценки по своему предмету
- Ученик - видит свой дневник и табель

## Как запустить локально
1. Установите зависимости: `pip install -r backend/requirements.txt`
2. Создайте файл `.env` с настройками PostgreSQL
3. Инициализируйте базу: `python backend/init_db.py`
4. Запустите сервер: `uvicorn backend.app.main:app --reload`
